Project: Leylix
Bhargav Vagadiya developed Leylix, a social media application built with Flutter, available on the Google Play Store. Bhargav served as Lead Developer on Leylix at Elaunch Solutions. Leylix is a comprehensive social networking platform featuring user profiles, posts, comments, likes, and social interactions. Bhargav implemented real-time chat and push notifications in Leylix using Socket.io integration. He integrated Firebase Authentication for secure user login in Leylix. He used Firebase Cloud Storage in Leylix for media management including photo and video uploads. He integrated Firebase Push Notifications in Leylix for user engagement and activity alerts. Bhargav applied the MVVM architecture pattern in Leylix for a clean and maintainable codebase.
