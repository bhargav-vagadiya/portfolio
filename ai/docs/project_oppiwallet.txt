Project: oppiwallet
Bhargav Vagadiya developed OppiWallet, a fintech digital wallet application built with Flutter, available on the Google Play Store. Bhargav served as Lead Developer on OppiWallet at Elaunch Solutions. OppiWallet is a secure fintech application featuring wallet management, payment processing, and full transaction history. Bhargav implemented biometric authentication and data encryption in OppiWallet for secure user access. He integrated RESTful APIs in OppiWallet for payment gateway integration and real-time transaction updates. Bhargav used the Drift database in OppiWallet for local data persistence and offline transaction support. OppiWallet follows MVVM architecture for a scalable and maintainable Flutter codebase.
