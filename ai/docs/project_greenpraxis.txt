Project: GreenPraxis App
Bhargav Vagadiya developed GreenPraxis, an environmental monitoring application built with Flutter. Bhargav served as Lead Developer on GreenPraxis at Tagline Infotech LLP. GreenPraxis is used for land data visualization and environmental field data collection. Bhargav integrated the Google Maps SDK in GreenPraxis with custom markers, polygons, and polylines for visualizing land and environmental data on maps. He implemented background data upload in GreenPraxis using Dart Isolates and Compute functions for efficient processing without blocking the UI. Bhargav developed resumable image upload functionality in GreenPraxis using the Tus.io library to handle large file uploads reliably. He used the Hive database in GreenPraxis for offline data storage and synchronization, allowing field workers to use the app without an internet connection.
